#  Court Outcome Prediction (NLP)
###A Machine Learning Project for Predicting Court Case Outcomes Using Legal Text

This project predicts whether a court case will be **affirmed** or **reversed** based on the written **facts** of the case.
Using Natural Language Processing (NLP) and machine learning, the system analyzes legal text and learns patterns in judicial reasoning to classify the final case outcome.
